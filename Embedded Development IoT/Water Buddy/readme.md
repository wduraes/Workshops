# Water Buddy

This sample shows the Water Buddy complete with IoT connection. 

It uses the kit you received which includes:

1. ESP8266 microcontorller
1. Breadboard
1. 1" OLED Screen
1. Soil Moisture sensor
1. 5V Water Pump
